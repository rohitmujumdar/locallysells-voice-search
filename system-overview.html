LocallySells Voice Search — System Overview

A Vapi voice agent that lets a caller ask "do you have Marlboro Reds?" and hear back real products and prices from the LocallySells tobacco catalog. This page maps how the pieces fit, and exactly what surface of LocallySells the agent touches.

The 30-second mental model

Vapi runs the live phone call (hears the caller, thinks with an LLM, speaks back). When the LLM needs real product info, it calls our one tool, search_products, which queries a copy of the LocallySells catalog and hands back the top matches with prices. Everything else about LocallySells (orders, payments, delivery, database) stays a black box the agent never touches.

Anatomy of one voice call

Listen → think → speak, with a detour to our tool when the agent needs data

The caller never knows a tool was called. To them it is one smooth conversation. The orange and purple parts are the only thing we built. Everything teal is configuration we handed to Vapi.

The tool contract

One tool, search_products — what goes in, what comes back

→ Vapi sends us (request)

// the LLM filled in these arguments
{
  "message": {
    "toolCallList": [{
      "id": "call_abc123",
      "function": {
        "name": "search_products",
        "arguments": {
          "query": "marlboro red",
          "category": "cigarettes"
        }
      }
    }]
  }
}

← We return (response)

{
  "results": [{
    "toolCallId": "call_abc123",
    "result": "Found 2 matches.
      Top results:
      - Marlboro Red King Size — $8.99 — Pack
      - Marlboro Red Carton — $84.99 — Carton"
  }]
}

The toolCallId must echo back so Vapi matches the answer to the question. The result is plain text the LLM reads, then it decides how to say it out loud (our system prompt tells it to summarize the top 3 and always include prices).

In production this same search runs as GET /api/master-catalog/search?q=&category=&limit= against Supabase. For the demo we copied the catalog to a CSV and replicated the search logic, so nothing depends on the live app being up on stage. The contract is identical, the agent would not know the difference.
